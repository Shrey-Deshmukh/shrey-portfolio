# Personal Portfolio

This repository contains my personal portfolio website, built to showcase my projects, experience, and leadership activities.

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- HTML5 & CSS3 (custom design system)
- Netlify (CI/CD & hosting)

## Features
- Responsive, mobile-friendly layout
- Clean, component-based architecture
- Performance-optimized styling
- Single-page navigation

## Deployment
The site is deployed on Netlify with continuous deployment enabled.  
Every push to the main branch triggers an automatic build and deploy.
